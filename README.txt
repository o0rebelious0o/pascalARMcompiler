Written in Gedit in C++ on Ubuntu Linux 12.04

Uses tools:-
******************************************
g++ (Ubuntu/Linaro 4.6.3-1ubuntu3) 4.6.3

Bison (GNU Bison) 2.5

Flex 2.5.35

Gramdiag 0.5a
*****************************************

To compile from souce:
	make
	
To run:

	./CW1 <soucefilename.pas>
	
e.g
	
	./CW1 sample1.pas
	
The Runnable ARM .s Assembly Code is contained in the file output.s
This ARM output has been verified using the ARM debugger tool
